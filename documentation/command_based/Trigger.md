`setType()` allows specifying whether the `Command` is ended when the trigger is no longer `true` (`atmt::WhileTrigger`) or whether it keeps running (default, `atmt::OnTrigger`). `setCriteria()` allows specifying another `Trigger` which must also be `true` for this `Trigger` to trigger. `inMode()` allows specifying which `TimedRobot` modes (Disabled, Autonomous, Teleop) the `Trigger` is able to trigger in.
